# 计算器程序 - 用户指南

## 简介

这是一个基于命令行的简易计算器程序，支持基本的四则运算（加、减、乘、除）。

## 系统要求

- Python 3.6 或更高版本
- 操作系统：Windows / macOS / Linux

## 快速开始

### 1. 运行程序

在终端中执行以下命令：

```bash
cd calculator/
python calculator.py
```

### 2. 使用示例

#### 基本运算

启动程序后，你会看到提示符，可以输入表达式进行计算：

```
=== 简易计算器 ===
支持的运算: +, -, *, /
输入格式: 数字 运算符 数字
输入 'q' 退出, 输入 'c' 清空

请输入表达式: 10 + 5
结果: 15.0

请输入表达式: 3.5 * 2
结果: 7.0
```

#### 连续运算

每次计算后结果会自动保存，可以在下次运算中使用：

```
请输入表达式: 100 - 30
结果: 70.0

请输入表达式: 70 / 2
结果: 35.0
```

#### 清空重置

输入 `c` 清空当前计算结果：

```
请输入表达式: c
已重置
```

#### 退出程序

输入 `q` 退出程序：

```
请输入表达式: q
感谢使用，再见！
```

## 运算符说明

| 运算符 | 功能 | 示例 |
|--------|------|------|
| `+` | 加法 | `3 + 5` → 8.0 |
| `-` | 减法 | `10 - 4` → 6.0 |
| `*` | 乘法 | `6 * 7` → 42.0 |
| `/` | 除法 | `15 / 3` → 5.0 |

## 错误处理

当输入有误时，程序会给出友好的提示信息：

| 错误情况 | 提示信息 |
|----------|----------|
| 格式错误 | 输入格式有误，请使用: 数字 运算符 数字 |
| 除数为零 | 错误：除数不能为零 |
| 无效数字 | 请输入有效数字 |
| 未知运算符 | 不支持的运算符: x，请使用 +, -, *, / |

## 注意事项

- 数字支持整数和小数（如 `3.14`）
- 运算符和数字之间需要用**空格**分隔
- 除法结果可能产生浮点数
