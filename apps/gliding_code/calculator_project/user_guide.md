# 计算器用户指南

## 1. 简介

Python 计算器是一个命令行交互式计算工具，支持基本算术运算和数学函数。程序采用友好的交互界面，便于日常使用。

## 2. 环境要求

- Python 3.8 或更高版本
- 无需安装第三方依赖包

## 3. 快速开始

### 3.1 运行计算器

```bash
cd calculator_project
python calculator.py
```

### 3.2 基本操作流程

1. 启动程序后，将显示欢迎信息和功能菜单
2. 根据菜单提示输入对应数字选择运算类型
3. 按提示输入操作数
4. 查看计算结果
5. 选择继续运算或退出程序

## 4. 功能说明

### 4.1 支持的运算

| 编号 | 运算     | 说明                     | 示例               |
|------|----------|--------------------------|--------------------|
| 1    | 加法     | 计算两个数的和           | 3 + 5 = 8          |
| 2    | 减法     | 计算两个数的差           | 10 - 3 = 7         |
| 3    | 乘法     | 计算两个数的积           | 4 × 5 = 20         |
| 4    | 除法     | 计算两个数的商           | 10 ÷ 2 = 5         |
| 5    | 幂运算   | 计算一个数的幂           | 2³ = 8             |
| 6    | 平方根   | 计算一个数的平方根       | √9 = 3             |
| 7    | 取模     | 计算两个数相除的余数     | 10 % 3 = 1         |
| 0    | 退出     | 退出计算器程序           | -                  |

### 4.2 输入说明

- **数字输入**: 支持整数（如 `42`）和浮点数（如 `3.14`）
- **运算选择**: 输入菜单项对应的数字（0-7）
- **退出程序**: 选择 `0` 或按 `Ctrl+C`

### 4.3 运算示例

```
--- 加法 ---
请输入第一个数字: 3.14
请输入第二个数字: 2.86

✅ 结果: 6.0
```

## 5. 错误处理

### 5.1 常见错误及处理方式

| 错误场景       | 提示信息                     | 处理方式         |
|----------------|------------------------------|------------------|
| 除数为零       | 除数不能为零                 | 重新输入非零除数 |
| 负数开平方     | 不能对负数开平方             | 输入非负数       |
| 无效数字输入   | 请输入有效的数字             | 重新输入数字     |
| 无效菜单选择   | 无效选择，请重新输入         | 重新选择         |

### 5.2 程序不会崩溃

所有用户输入错误都会被优雅捕获并给出明确提示，程序会持续运行直到用户主动退出。

## 6. 运行测试

### 6.1 安装 pytest

```bash
pip install pytest
```

### 6.2 执行测试

```bash
cd calculator_project
pytest test_calculator.py -v
```

### 6.3 测试覆盖

测试覆盖以下场景：
- 基本运算功能测试（加、减、乘、除、幂、平方根、取模）
- 边界条件测试（零、负数、大数、极小浮点数）
- 异常场景测试（除零、无效输入等）
- 数学性质验证（交换律、结合律）

## 8. 扩展开发

如需添加新的运算功能：

1. 在 `Calculator` 类中添加新的计算方法
2. 在 `CalculatorCLI.MENU_ITEMS` 中添加菜单项
3. 在 `test_calculator.py` 中添加对应的测试用例
